## Overview

- Built with [gatsby](https://github.com/gatsbyjs/gatsby).

## Development

```bash
# Install dependencies.
yarn

# Start dev server.
yarn dev
```

## Build

```bash
# Build to /public
yarn build
```

## Deploy

Deployment is done using Netlify.
When changes are merged into master the site will be deployed.
